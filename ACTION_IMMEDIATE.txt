ÉTAPE 1 : TESTER QUE LE RESPONSIVE FONCTIONNE
────────────────────────────────────────────────

Double-cliquez sur : test-simple.html

Ce fichier va vous montrer si le CSS responsive fonctionne.

✅ Si les boîtes passent de 3 → 2 → 1 colonne quand vous 
   redimensionnez = Le responsive FONCTIONNE !

❌ Si rien ne change = Problème de navigateur


ÉTAPE 2 : VIDER LE CACHE D'INDEX.HTML
────────────────────────────────────────────────

Ouvrez index.html et appuyez sur :

▶ Windows : Ctrl + Shift + R
▶ Mac     : Cmd + Shift + R

OU

▶ F12 → Onglet Network → Cochez "Disable cache" → F5


ÉTAPE 3 : TESTER AVEC CHROME DEVTOOLS
────────────────────────────────────────────────

1. Ouvrez index.html
2. Appuyez sur F12
3. Cliquez sur l'icône 📱 en haut à gauche
4. Sélectionnez "iPhone 12" ou entrez "375px"
5. Vous DEVEZ voir :
   - Menu hamburger (3 barres oranges)
   - Services en 1 colonne
   - Textes adaptés
